# Dwellot Website Maintenance Developer Role
## Budget: $250/month (GH₵4,000-4,500)

---

## Role Overview
**Position**: Part-Time Website Maintenance Developer  
**Time Commitment**: 12-15 hours/month (~3-4 hours/week)  
**Rate**: GH₵300-350/hour (~$18-22/hour equivalent)  
**Contract Type**: Monthly retainer with renewable agreement  
**Location**: Remote (Ghana-based preferred for timezone alignment)

---

## Required Skills
### Must Have:
- 1-2+ years experience with Next.js and React
- TypeScript proficiency
- Supabase/PostgreSQL knowledge
- Git/GitHub workflow
- Basic understanding of SEO principles
- Responsive design (Tailwind CSS)

### Nice to Have:
- Real estate/property platform experience
- Vercel deployment experience
- Basic DevOps knowledge
- UI/UX sensibility

## Monthly Responsibilities Breakdown

### Week 1: Monitoring & Updates (3-4 hours)
**Tasks:**
- [ ] Review site performance metrics (Vercel Analytics)
- [ ] Check for broken links or 404 errors
- [ ] Monitor Supabase database health
- [ ] Review error logs and fix critical bugs
- [ ] Update dependencies if needed (security patches)
- [ ] Test all forms (contact, inquiry, newsletter)
- [ ] Verify all property listings display correctly

**Deliverables:**
- Weekly status report with any issues found
- Bug fixes for critical issues

---

### Week 2: Content & Property Management (3-4 hours)
**Tasks:**
- [ ] Add new property listings (3-5 properties)
  - Write SQL scripts for property insertions
  - Optimize and add property images
  - Verify all property details display correctly
- [ ] Update existing property information as needed
- [ ] Archive or remove sold/rented properties
- [ ] Update homepage featured properties rotation
- [ ] Add/update blog content if provided

**Deliverables:**
- New properties live on site
- Updated property database
- Content changes documented

---

### Week 3: SEO & Performance (2-3 hours)
**Tasks:**
- [ ] Update sitemap with new properties/pages
- [ ] Monitor Google Search Console for errors
- [ ] Check page load speeds and optimize if needed
- [ ] Review and update meta descriptions for key pages
- [ ] Ensure all images have proper alt text
- [ ] Fix any SEO warnings from audits
- [ ] Submit sitemap to Google if major changes made

**Deliverables:**
- SEO health report
- Performance improvements implemented
- Search visibility maintained/improved

---

### Week 4: Feature Enhancements & Planning (2-3 hours)
**Tasks:**
- [ ] Implement 1 small feature or improvement per month
  - Examples: New filter, UI enhancement, email template
- [ ] Code review and refactoring of problematic areas
- [ ] Security audit of forms and user inputs
- [ ] Plan next month's priorities
- [ ] Documentation updates

**Deliverables:**
- 1 new feature/improvement shipped
- Monthly summary report
- Next month's task list
